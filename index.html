qambi
headless MIDI sequencer for your browser

Headless means that there is no GUI; qambi is intended to be used as an engine behind your application. You can make any kind of application on top of qambi; a game, an online DAW, artistic sites, music science experiments and so on.

Code is available at Github and published under MIT license. You can add qambi to your project by installing it via npm npm install qambi.

Qambi is a rebuild (in progress) of heartbeat, documentation is still very sparse but I am working on it. Should you have any questions or requests, please don't hesitate to drop me a line or file an issue.
