qambi

A headless MIDI sequencer for your browser written in es2016 (MIT).

Headless means that there is no GUI; qambi is intended to be used as an engine behind your application. Qambi is set up very flexible so you can make any kind of application on top of it; a game, an online DAW, artistic sites, music science experiments and so on. Qambi is a rebuild (in progress) of heartbeat, documentation is in progress as well, see the wiki.
