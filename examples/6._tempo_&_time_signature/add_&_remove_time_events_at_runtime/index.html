You can add tempo and time signature events by clicking the "add" button.

You can change the values of bpm, nominator and denominator in the input fields.

The time event will be inserted at the position of the playhead.

You can add or remove time events while the song is playing.

You can remove a time event by selecting a time event in the drop down menu and then click the "remove" button.

The MIDI standard allows you to add multiple events at the same position. By clicking the button "remove double time events" you can remove double tempo and/or time signature events. Only the lastly added event of the same type will be kept. For instance if you add 2 tempo events at 2.1.1.0, the first with bpm 123 and the second with 125, after clicking the button the tempo event with bpm 123 will be removed.
