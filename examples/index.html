qambi

Below a few live examples of how you can use qambi. Documentation is sparse but in progress, see the wiki. Should you have any questions or requests, please don't hesitate to drop me a line or file an issue.

Song

  • Create a song from a MIDI file and play it back using City Piano, a qambi sample instrument with 4 velocity layers.
  • Create song from scratch and save it as MIDI file.

MIDI i/o

  • Play instruments using your MIDI keyboard
  • Record MIDI
  • Playback a MIDI file via both an internal instrument and an external softsynth

Miscellaneous

  • Change the pitch of a song
